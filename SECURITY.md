Security Policy

Supported versions:

As this is a single script project under active development, only the latest release is supported with security updates.
- latest version is always supported.
- older releases are not supported.

Reporting a vulnerability:

If you discover a security vulnerability in this project, please do not open a public GitHub issue. 
Instead, use GitHub's private vulnerability reporting feature so it can be reviewed and addressed before any public disclosure.
https://github.com/VibedOut/Outlook-Inbox-Sorter/security/advisories/new

Please include as much detail as possible.
- a description of the vulnerability
- steps to reproduce
- potential impact

I will aim to respond within 5 business days, and will keep you updated as the issue is investigated. 

IMPORTANT NOTES:

This script connects to a M365 mailbox via Outlook COM, and handles email data. Users are responsible for ensuring it is deployed in accordance with their organizations security policies.

**No credentials or tokens are stored by this script.**
